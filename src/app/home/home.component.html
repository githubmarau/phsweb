Seja bem vindo ao PH Sistem WEB


Este software foi desenvolvido conforme a necessidade de uma integração mobile com o software já existente em nossos clientes.

Se você já estiver visualizando este tutorial, quer dizer que já se logou ao software.

Primeiramente deve saber que, como cliente as funções de relatório estarão relacionadas à conexão com o banco de dados da loja. Isso quer dizer que para acessar os dados da loja você precisa escolher antes a que loja deseja se conectar.

Para se conectar aos dados de uma loja vinculada ao teu usuário, você deve clicar em cima do teu nome, localizado no canto superior direito e depois em Perfil.

Aparecerá uma tela com os dados do usuário logado no canto superior esquerdo, e uma lista contendo os nome das lojas que possivelmente você poderá se conectar.

Após escolher os dados da loja a que deseja você poderá clicar na opção Dashboard no menu a esquerda da tela.
